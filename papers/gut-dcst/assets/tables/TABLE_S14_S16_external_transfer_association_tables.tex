\section*{Supplementary External Validation Transfer Tables}

The tables below summarize direct transfer of AGP-derived absorb dCST labels to the clinically annotated IBD cohorts emphasized in the main text. Frequentist columns report Fisher odds ratios from the sample-level 2x2 tables used in the frozen-transfer validation. Bayesian columns report the corresponding Jeffreys-prior posterior median odds ratio and 95\% credible interval derived from the same case/control counts. The rendered tables are unfiltered; the same full machine-readable tables are also written alongside these manuscript assets as TSV files.
