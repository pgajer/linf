## Stage 7. Establish the ongoing incremental-integration rule

Objective:
Prevent the manuscript from drifting back into a merged or exploratory state.

Rule for every new result:

Before integration, answer these questions in writing:

1. What is the claim?
2. What is the strongest caveat?
3. Does this belong in the main text, supplement, or notes?
4. What existing text must be shortened or removed if this enters the paper?

If those answers are not clear, the result stays out of the manuscript for now.

Done criterion:

- Future additions arrive by replacement or sharpening, not by accumulation.
